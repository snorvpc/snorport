ゲーム攻略・作業効率化のための自作ツール集。
ファラウェイをはじめ、日々をちょっと楽にするものを作っています。

PUBLISHED TOOLS

Faraway Faraway Tracker ファラウェイ向けのトラッカーツール。
ツールを開く GitHub
https://snorvpc.github.io/faraway-tracker/
AZUL AZUL タイルトラッカー AZULのタイル枚数をタップでカウント。5ラウンド目の出現タイルを自動予測。
ツールを開く GitHub
https://snorvpc.github.io/azul/
Papayoo パパヨー カウンター パパヨー専用カード追跡ツール。スート別・パヨー別に出済み/パス管理、5Rの残りスコアを自動計算。
ツールを開く GitHub
https://snorvpc.github.io/papayoo/
Nimmt ニムト カード管理 Nimmtのカード使用状況をタップ・ドラッグで管理。残りの牛の頭スコアをリアルタイム表示。
ツールを開く GitHub
https://snorvpc.github.io/nimmt/
